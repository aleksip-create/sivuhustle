Mitä teet seuraavaksi

Noin 20 minuuttia. Järjestys on tärkeä.

→ Päivittäinen aamuohje (5 min)
1

Julkaise muutokset Verceliin

Uudet sivut (kiitos-lataus, monetisointi) eivät näy ennen deployta.

  1. Avaa vercel.com/dashboard
  2. Valitse SivuHustle-projekti → Deployments
  3. Jos Git on kytketty: puske tiedostot repoon → deploy käynnistyy automaattisesti
  4. Jos ei Gitiä: Settings → Git yhdistä repo TAI käytä konetta jossa on Node → npx vercel --prod

Tarkista deployn jälkeen: kiitos-lataus.html avautuu (ei 404).

2

Supabase: UTM-kentät leadeille

Näet mistä Meta-mainos toi leadin.

  1. Supabase SQL Editor → New query
  2. Kopioi alla oleva SQL → Run
alter table public.leads add column if not exists utm_source text;
alter table public.leads add column if not exists utm_medium text;
alter table public.leads add column if not exists utm_campaign text;
alter table public.leads add column if not exists utm_content text;
alter table public.leads add column if not exists landing_page text;
create index if not exists leads_utm_campaign_idx on public.leads (utm_campaign);
3

Automaattinen tervetulosähköposti (Resend)

Edge Function on jo deployattu. Tarvitsee vain API-avaimen.

  1. Luo tili resend.com → API Keys → luo avain
  2. PowerShell projektikansiossa:
    supabase secrets set RESEND_API_KEY=re_...
  3. (Myöhemmin) Vahvista domain sivuhustle.fi Resendissä → lähettäjä asiakaspalvelu@sivuhustle.fi

Ilman avainta leadit tallentuvat normaalisti — sähköposti vain ohitetaan.

4

Testaa putki (5 min)

  • ☐ Avaa ilmainen opas -lomake → täytä testisähköposti
  • ☐ Ohjautuu kiitos-lataus.html -sivulle
  • ☐ Supabase → Table leads → uusi rivi
  • ☐ Meta → Events Manager → Testitapahtumat → Lead
  • ☐ GA4 → Realtime → generate_lead
5

Meta-mainos — jatka nykyistä + remarketing myöhemmin

Nyt: pidä cold-mainos 12 €/pv vielä 3–5 päivää. Mittaa CTR ja Leadit.

Kun 20+ Lead-tapahtumaa: käynnistä Kampanja 2 → meta-kampanja.html (violetti Remarketing-osio).

  • • Hyvä: alle 2 € / Lead
  • • Nosta budjettia vain jos Leadit tulevat
  • • CTR < 0,8 % 3 pv → vaihda mainostekstiä
Jo valmiina (ei tarvitse tehdä):
  • ✓ GA4 + Meta Pixel sivustolla
  • ✓ Stripe Pro / vuosi / Lifetime
  • ✓ send-lead-email Edge Function deployattu
  • ✓ Oppaiden Pro-CTA:t + kiitos-sivun upsell
