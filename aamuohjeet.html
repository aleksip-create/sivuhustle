Meta-mainos — mitä teet tänään?

Jos alle 20 Lead-tapahtumaa pixelissä:

Pidä nykyinen cold-mainos käynnissä 12 €/pv. Älä nosta budjettia ennen kuin Leadit tulevat tasaisesti.

Kun 20+ Lead-tapahtumaa:

Käynnistä remarketing (Kampanja 2). Ohjeet: meta-kampanja.html

Jos Lead > 3 € kolme päivää:

Pysäytä mainos hetkeksi. Tarkista landing (ilmainen opas -lomake) ja mainosteksti. Testaa uusi otsikko.

Affiliate-tulot (15 min tällä viikolla)

Gumroad-linkki on jo valmiina. Muut odottavat hakemustasi — tämä ei voi tapahtua automaattisesti.

  1. Avaa affiliate-liity.html → hae Hostinger, Canva, MailerLite
  2. Kun saat linkit → liitä js/config.js → AFFILIATE.[nimi].url
  3. Push GitHubiin → affiliate-linkit aktivoituvat koko sivustolla

Sähköposti leadeille (kerran)

Edge Function on deployattu. Tarvitsee vain Resend API -avaimen.

  1. Luo avain resend.com
  2. PowerShell: supabase secrets set RESEND_API_KEY=re_... (projektikansiossa tai C:\temp\sivuhustle-db)
  3. Testaa: täytä ilmainen opas -lomake → tarkista sähköposti

Ilman avainta leadit tallentuvat normaalisti — vain automaattinen tervetulosähköposti jää pois.
